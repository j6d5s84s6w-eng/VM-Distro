2

Fill it in

Open the template in Excel (preferred) or Numbers and complete each tab below. Every tab feeds into the final distro output — don't skip any.

New Products — codes & colors
  • Enter the product code for each new device being launched (e.g. iPhone 16, MacBook Pro 14)
  • Add all available colors/finishes for each product
  • Include any NPI (New Product Introduction) items — these drive the fixture count calculations
  • Do not include carry-over products that are not changing this cycle
Per-Fixture Quantities — how many devices per fixture, per color
  • For each product + color combination, enter how many units go on a single fixture
  • Reference your merchandising strategy to determine quantities per fixture type (e.g. 10ft table vs 12ft table)
  • Different cohorts (table-size ranges) may have different quantities — enter a value for each cohort that applies
  • Leave blank if a product does not appear on a given fixture type
Avenue Quantities — bento allocations
  • Enter the bento allocation for each product per avenue (e.g. Avenue 1, Avenue 2, etc.)
  • Allocations should reflect the approved merchandising plan from your GEO VM stakeholder
  • Each row represents one store avenue — confirm avenue count from the Accessories GSL
  • Zero-fill any avenues where a product is not allocated
Programs — overrides for specific stores
  • List any stores that require special handling — T@A HP, Boardrooms, Flagship stores, etc.
  • Enter the store number and the specific override (e.g. different quantity, different product mix)
  • Program overrides take priority over standard cohort calculations
  • Leave this tab empty if there are no store-specific exceptions this cycle

Configure & forecast

Set cohort units, buffer %, and forecast period. The output uses the wide WWRFO format (Grand Total with Buffer, Buffer, Product Zone Tables & Walls, Equipment Lists, Accessories Fixtures, Services and Avenues) with a TOTAL row at the top.

Buffer Rules Add a row per buffer rule. "Applies to" accepts table sizes (10, 7, 8, 12+) or fixture names (Watch Discovery, Watch Combined). "Per" controls whether qty is added to each color (4×) or once per table.
Cohorts One row per cohort. Table sizes accept any mix: 10, 8-12, 12+, ≤10, or comma lists like 6, 8-12, 14+.
