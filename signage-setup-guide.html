2×3 Signage Skill

Setup & Usage Guide — VM Team

Part 1 — One-Time Setup

1

Make sure Claude Code is installed

Open your terminal and type claude --version. If you see a version number, you're good. If not, install Claude Code first.

2

Get the skill file from Omaira

You'll receive a file called signage.md — save it somewhere you can find it (Desktop is fine for now).

3

Place it in the Claude commands folder

Open Finder, then in the menu bar click Go → Go to Folder (or press Shift + Cmd + G).

Type ~/.claude/commands/ and press Enter.

Drag the signage.md file into this folder.

Folder doesn't exist? Go to ~/.claude/ first, create a new folder called commands, then drop the file in.
4

Make sure you have the Python script

You'll need apply_languages_new.py saved on your Mac. Ask Omaira for the file path if you're unsure where it lives on your machine.

5

Restart Claude Code

Close and reopen Claude Code — it picks up new skills on startup.

Part 2 — Using the Skill

1

Type /signage in Claude Code

The skill will load and prompt you to fill in your variables.

2

Fill in the variables

Fixture Count File
Full path to your WW fixture count Excel file
/Users/you/Desktop/JUN26_Fixture.xlsx
WW Fixture Counts Tab
Exact name of the tab in the file
WW Fixture Counts JUN26
Country / Store Override
Any rule changes for this run — leave blank if none
omit if no changes
Output File Name
What to name the generated output file
JUN26_Signage_Output
Primary Language Column
Before running the skill, manually add a column called "Primary Language" to your Fixture Count file
Add to Fixture Count file first
Secondary Language Column
Before running the skill, manually add a column called "Secondary Language" to your Fixture Count file
Add to Fixture Count file first
China Division Note: Stores in China are automatically split into two divisions — Shanghai and Beijing. Make sure your GLS sheet includes the China division column so the lookup works correctly.
3

Hit Enter — Claude does the rest

Claude will run apply_languages_new.py with your inputs and generate the output file automatically.
